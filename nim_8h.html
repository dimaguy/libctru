Detailed Description

NIM (network installation management) service.

This service is used to download and install titles from Nintendo's CDN.

We differentiate between two different kinds of downloads:

  • "active" downloads, which are downloads started with NIMS_StartDownload or NIMS_StartDownloadSimple. The process must keep polling the current status using NIMS_GetProgress.
  • "tasks", which are downloads started with NIMS_RegisterTask. These are only processed in sleep mode.
